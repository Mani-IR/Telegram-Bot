# ربات تست تلگرام (Telegram Test Bot)

این یک ربات تلگرامی ساده نوشته‌شده با پایتون است که عمدتاً برای پاسخ‌دهی به پیام‌ها در گروه‌ها طراحی شده. ربات فقط وقتی فعال می‌شود که یک ادمین مجاز به یک پیام ریپلای کند – در این صورت، ربات به همان پیام ریپلای می‌کند و محتوای ریپلای ادمین را همراه با پیام اصلی نمایش می‌دهد.
این ربات مناسب برای مدیریت گروه‌ها، پاسخ به سوالات کاربران، یا سیستم پشتیبانی ساده است.

## ویژگی‌های اصلی

* پاسخ به دستورات /start و /help با پیام خوش‌آمدگویی.
* هندل کردن ریپلای‌های ادمین‌های مجاز در گروه‌های مشخص‌شده.
* اکو کردن پیام ریپلای‌شده ادمین به همراه متن پیام اصلی (برای تأیید یا پاسخ عمومی).
* محدود کردن دسترسی: فقط یوزرنیم‌های مشخص‌شده در لیست ادمین و فقط در چت‌های (گروه‌های) مجاز فعال است.
* استفاده از polling برای دریافت آپدیت‌ها.


## ساختار پروژه

``` 
src/
├── bot.py         # کد اصلی ربات (کلاس Testbot و هندلرها)
├── config.py      # تنظیمات حساس مثل توکن بات، لیست ادمین‌ها و چت‌های مجاز
├── constants.py   # پیام‌های ثابت و تنظیمات عمومی (مثل پیام خوش‌آمدگویی)
env/               # فایل محیط مجازی (virtual environment) - برای مدیریت وابستگی‌ها
```

## پیش‌نیازها

* نسخه Python 3.7 یا بالاتر
* کتابخانه pyTelegramBotAPI (معروف به telebot)

## نصب و راه‌اندازی

1. کلون کردن ریپو :

```
git clone https://github.com/yourusername/your-repo-name.git
cd your-repo-name
```
2. ایجاد و فعال کردن محیط مجازی (اختیاری اما توصیه‌شده) :

```
python -m venv env
env\Scripts\activate     #   در ویندوز
source env/bin/activate  # در لینوکس/مک

```

3. نصب وابستگی‌ها:

```
pip install pyTelegramBotAPI
```

4.  تنظیمات (`config.py`)

قبل از اجرای ربات، مقادیر زیر را در فایل `config.py` تنظیم کنید:

- **`BOT_TOKEN`**  
  توکن ربات را از ربات [@BotFather](https://t.me/BotFather) دریافت کرده و در این متغیر قرار دهید.

- **`ADMINS_USERNAMES`**  
  لیست یوزرنیم ادمین‌های مجاز (بدون `@`).
  ```python
  ADMINS_USERNAMES = ["------"]
  ```


5. اجرای ربات:

`python` src/bot.py



## نحوه کار در گروه

- ربات را به گروه اضافه کنید و ادمین کنید (حداقل حقوق ارسال پیام).
- وقتی کاربری سوالی می‌پرسد، یک ادمین مجاز (از لیست) به پیام او ریپلای کند.
- ربات به همان پیام ریپلای می‌کند و چیزی شبیه این نمایش می‌دهد:
```
✅ You replied to: 'متن پیام اصلی کاربر'

    Your message:   'متن پاسخ ادمین'
```
## نکات بهبود برای آینده

- اضافه کردن دیتابیس برای ذخیره پیام‌ها.
- استفاده از async (aiogram) برای عملکرد بهتر.
- مدیریت خطاها و لاگ‌گیری پیشرفته‌تر.
- اضافه کردن دستورات بیشتر مثل /stats یا مدیریت ادمین‌ها.
