### توضیح فایل‌ها (همه چیز با یک نگاه!)

اینجا دقیقاً توضیح دادم هر فایل چه کاری انجام می‌دهد تا خیلی راحت متوجه شوید:

**src/bot.py** → فایل اصلی ربات (همین فایل را اجرا می‌کنید)  
• ربات را راه‌اندازی می‌کند  
• دستورات /start و /help را مدیریت می‌کند (پیام خوش‌آمدگویی می‌فرستد)  
• وقتی ادمین مجاز در گروه مجاز به پیام کسی ریپلای کند، ربات پاسخ ادمین را همراه با متن پیام اصلی تأیید می‌کند  
• همه هندلرها (دستورات) اینجا تعریف شده‌اند  
• ساختار تمیز با کلاس Testbot برای خوانایی بیشتر

**src/config.py** → فایل تنظیمات  
• توکن ربات (BOT_TOKEN)  
• لیست ادمین‌های مجاز (ADMINS_USERNAMES) — فقط این افراد می‌توانند ریپلای کنند  
• لیست گروه‌های مجاز (VALID_CHATS) — ربات فقط در این گروه‌ها کار می‌کند  
• برای اجرا فقط کافیست مقادیر را عوض کنید

**src/constants.py** → فایل پیام‌های ثابت  
• پیام خوش‌آمدگویی، پیام در حال جستجو و حداکثر طول پاسخ  
• اگر بخواهید متن‌های ربات را تغییر دهید، اینجا عوض کنید

### فایل اصلی برای اجرا
`src/bot.py`  
فقط کافیست در ترمینال بنویسید:  
```bash
python src/bot.py
```
### ربات دقیقا چیکار میکنه ؟ 


1. وقتی کسی در چت خصوصی یا گروه دستور /start یا /help بزند → «سلام به ربات تست خوش آمدید!» می‌فرستد
2. وقتی کاربری در گروه سوال می‌پرسد و ادمین مجاز (مثلاً شما) به پیام او ریپلای می‌کند →
ربات به همان پیام ریپلای می‌کند و می‌نویسد:
```
✅ You replied to: 'سوال کاربر'

Your message: 'جوابی که شما نوشتید'
```
